Image Size

Image size is the per-character crop resolution used before matching. Larger values can improve quality, but they also take longer.

Higher resolutions increase runtime. Start with 64x64 and increase only when needed.

Image Size: 64x64
Allowed range per side: 32 to 256.

Expected Characters (Optional)

If provided, export will compare detected count to this value and show a warning when they differ.

Characters Per Row (Optional)

If provided, detected crops are ordered with this row size before matching. Leave empty to use the default behavior (5 per row).

Batch Upload Screenshots

Run the same export settings on many screenshots in sequence and merge everything into one favorites JSON file. The image size, expected count, characters per row, and type/class filters above are reused for every file.

The browser controls the download destination. In most setups this means the default Downloads folder.
